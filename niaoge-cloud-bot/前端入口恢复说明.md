# 前端入口恢复说明

如果后续需要恢复之前隐藏/注释掉的功能入口，按以下步骤操作后重新打包即可。

## 1. 恢复底部导航"主页"入口

文件：`frontend/src/components/BottomNav.vue`

取消主页入口注释，并把"账号"和"设置"保持打开：

```vue
<router-link to="/" class="nav-item" exact-active-class="active">
  ...
  <span>主页</span>
</router-link>
<router-link to="/accounts" class="nav-item" active-class="active">
  ...
  <span>账号</span>
</router-link>
<router-link to="/setting" class="nav-item" active-class="active">
  ...
  <span>设置</span>
</router-link>
```

## 2. 恢复首页路由

文件：`frontend/src/router/index.js`

将 `/` 从 redirect 改回 Home.vue 组件：

```js
// { path: '/', redirect: '/accounts' },
{ path: '/', name: 'home', component: () => import('../views/Home.vue') },
```

并把登录成功跳转改回 `/`：

```js
} else if (token && to.path === '/login') {
  next('/')
}
```

## 3. 恢复底部导航"任务"入口

文件：`frontend/src/components/BottomNav.vue`

将注释掉的代码取消注释：

```vue
<router-link to="/tasks" class="nav-item" active-class="active">
  ...
  <span>任务</span>
</router-link>
```

## 4. 恢复 `/tasks` 和 `/batch` 路由

文件：`frontend/src/router/index.js`

取消以下两行注释：

```js
{ path: '/tasks', name: 'tasks', component: () => import('../views/Tasks.vue') },
{ path: '/batch', name: 'batch', component: () => import('../views/Accounts.vue') },
```

## 5. 恢复"功能中心"的批量操作入口

文件：`frontend/src/views/Func.vue`

找到"批量操作"那一行，改回 `/batch`：

```vue
<div class="list-item" @click="$router.push('/batch')">
  <div class="list-left"><span>🚀 批量操作</span></div>
  <span style="font-size:13px;color:var(--text-muted)">副本 · 爬塔 · 资源 · 商店 →</span>
</div>
```

## 6. 恢复 Home.vue 快捷操作

文件：`frontend/src/views/Home.vue`

- 取消"一键连接"按钮注释。
- "执行每日"改回直接执行：`@click="doAction('daily-all')"`。
- "批量操作"按钮取消注释（指向 `/tasks`）。

## 7. 重新打包

```bash
cd frontend
npm run build
```

打包完成后刷新 `http://localhost:3456` 即可看到恢复后的入口。
